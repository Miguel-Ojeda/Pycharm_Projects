<tag>TEXTO</tag>    o sea, TEXTO es lo que hay entre un mismo tag, entre la abertura y el cierre
ELEMENTO es todo el pack entre la apertura y el cierre del tag...

Ejemplos... de elementos:
<tag1>TEXTO!</tag1>  ---> todo esto llamamos elementos, está formado por el tag (apertura y cierre) y el texto que está
en medio!!

Obviamente, los tags para HTML están definidos, no pueden ser cualquier cosa, son cosas definidas en el lenguaje
(a diferencia de lo que ocurre con el XML que es extensible=

Los tags pueden ir con atributos... se indican con ---> ATRIBUTO="valor del atributo"
Por ejemplo, el tag a se utiliza para denotar un hiperenlace.... lleva un atributo que indica cuál es la URL
Al's free <a href="https://inventwithpython.com">Python books</a>.
Aquí el elemento sería <a href="https://inventwithpython.com">Python books</a>.
los tags son del tipo <a> (apertura y cierre)
El texto sería Python book
El tag a tiene un atributo href cuyo valor es "http........"
